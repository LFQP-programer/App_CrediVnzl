## ?? Flujos de Navegación

### Flujo 1: Modificar desde Lista
```
Lista de Clientes
    ? (swipe left)
[?? Editar]
    ?
Editar Cliente
    ? (Guardar Cambios)
Lista de Clientes ? (actualizada)
```

### Flujo 2: Modificar desde Detalle
```
Lista de Clientes
    ? (tap en cliente)
Detalle Cliente
    ? (botón Modificar)
Editar Cliente
    ? (Guardar Cambios)
Detalle Cliente ? (actualizado)
```

### Flujo 3: Eliminar desde Lista
```
Lista de Clientes
    ? (swipe left)
[??? Eliminar]
    ?
?? Confirmación
    ? (Sí, eliminar)
Lista de Clientes ? (sin el cliente)
```

### Flujo 4: Eliminar desde Detalle
```
Lista de Clientes
    ? (tap en cliente)
Detalle Cliente
    ? (botón Eliminar)
?? Confirmación
    ? (Sí, eliminar)
Lista de Clientes ? (sin el cliente)
```
