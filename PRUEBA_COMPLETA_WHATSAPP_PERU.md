## ?? SOLUCIÓN DE PROBLEMAS

### Problema 1: WhatsApp no se abre

**Posibles causas:**
- WhatsApp no instalado
- Número de teléfono inválido
- Permisos de la app

**Solución:**
```
1. Verificar que WhatsApp esté instalado
2. Verificar formato del número: 51XXXXXXXXX
3. Intentar con "No, solo copiar"
4. Verificar logs en Output de Visual Studio
```

### Problema 2: Número no tiene formato peruano

**Posibles causas:**
- WhatsAppService no convierte correctamente
- Número no empieza con 9

**Solución:**
```
1. Verificar WhatsAppService.LimpiarNumeroTelefono()
2. Verificar que número empiece con 9
3. Limpiar y reconstruir proyecto
```

### Problema 3: Credenciales no funcionan

**Posibles causas:**
- Contraseña mal copiada
- Usuario no creado en BD
- Hash incorrecto

**Solución:**
```
1. Copiar contraseña exactamente como aparece
2. Verificar usuario en "Usuarios Activos"
3. Regenerar contraseña si es necesario
4. Verificar logs de autenticación
```

### Problema 4: Login del cliente falla

**Posibles causas:**
- DNI incorrecto
- Contraseña incorrecta
- Usuario no activo

**Solución:**
```
1. Verificar que DNI sea exactamente el mismo
2. Verificar que contraseña sea exactamente la misma
3. Verificar en BD que usuario esté activo
4. Verificar que Estado = Activo
```
